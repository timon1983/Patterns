Шаблон проектирования Итератор(Iterator).
-Интерфейс Iterator предоставляет два метода , hasNext() для определения наличия следующего элемента и next() для
получения следующего элемента , если он есть.
-Реализуется этот интерфей внутренним классом WorkerIterator в классе Worker.
-Интерфейс Collection предоставляет метод для получения объекта типа Iterator и его реализует класс Worker.
