Шаблон проектировния Приспособленец(Flyweight).
-Классы Mason и Roofer реализуют интерфейс Worker c методом doWork().
-В классе WorkFactory создются объекты Mason и Roofer и записываются в Map.
-В классе клиенте мы можем получить эти объекты с помощью метода getNewWorke() и дать им команду на выполнение работы
множество раз.
