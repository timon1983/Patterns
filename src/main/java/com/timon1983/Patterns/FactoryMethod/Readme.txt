                Шаблон проектирования Фабрика(Factory Method)
-С помощь данного паттерна реализован упрощенный вид программы строительства дома.
-По мере необходимости вызываются работник для производства работ над объектом который мы указываем в аргументе
статического метода getWorkerFactoryByWorkObject(String workObject).
-Создаются четыре класса работников Mason, Roofer, Plumber и Electrician которые реализуют интерфейс Worker где
содержится единственный метод для команды на выполнение своей работы doWork().
-Для создания рабочих созданы четыре класса MasonFactory, RooferFactory, ElectricianFactory и PlumberFactory которые
релазуют интерфейс WorkerFactory где содержится один метод для создания и возвращения объекта типа Worker.
