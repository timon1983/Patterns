Шаблон проектирования команда(Command).
-Класс Worker принимает в конструктор объекты выполняемых задач строительной организации BuldCommand, CheckCommand,
ProjectCommand тем самым инкапсулируя запрос на выполнение задач через них.
-Эти классы реализуют интерфейс Work , а также принимают в конструктор объект BuildingOrganization , в классе
которого используют методы для реализации метода doWork().
