Шаблон проектирования Цепочка ответственности(ChainOfResponsibiliti).
-Программа имитирует отправку и получение посылки почтой.
-Классы DeliveryNotification, DeliveryPackages, DeliveryService наследуют абстрактный класс PostOffice, в котором
с помощью метода setPostOffice() устанавливается очередность цепочки обработки событий классов наследников.
-С помощью метода mailOperation() выполняется обработка события и проверка на наличия следующего звена цепочки.
