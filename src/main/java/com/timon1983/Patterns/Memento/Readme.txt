Шаблон проектирования Хранитель(Memento).
-Объекты класса ArhiveBase хранят в себе состояние объекта WorkPlan  после вызова им метода setArhive().
-Можно создать множество объектов ArhiveBase для хранения состояния объекта WorkPlan в определенный период.
-Вернуть сохраненное состояние можно при помощи метода load() , указав в аргументе объект класса ArhiveBase
который хранит в себе искомое состояние объекта WorkPlan.
