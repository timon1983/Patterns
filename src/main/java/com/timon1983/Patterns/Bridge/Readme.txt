Шаблон проектирования Мост(Bridge).
-Классы Mason и WoodCraftsman реализуют интерфейс Worker , который задает один метод doWork() на выполнение работы
по строительству дома.
-Классы StoneHouse и WoodenHouse наследуются от абстрактного класса Building , реализуя метод build() на типа дома
который нужно построить, а также принимают объект Worker , т.е. можно выбрать рабочих которые будут заниматься стройкой.
