Шаблон проектирования Интерпритатор(Interpreter).
-Классы HandyMan и TypycalWorker определяют какие работники нужны, местера на все руки или обычные работники,
по контексту который задает клиент.
-В классе WorkerSkill метод interpret() определяет совпадают ли навыки рабочего с заданым контекстом.
-В методах  interpret() классов HandyMan и TypycalWorker определяют подходят ли набор навыков для данного рабочего.
