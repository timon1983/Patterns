Шаблон проектирования Builder.
-C помощью данного шаблона мы реализовали программу для заказа автомобиля.
-Абстрактный класс CarsBuilder имеет абстрактные методы для задания параметров автомобиля.
-Классы CarForGazprom и CarForYandexDrive являются наследниками CarsBuilder и задают парметры автомобиля согласно
требованиям заказчика , реализуя методы класса CarsBuilder.
-С помощью класса CarManager принимается заказ от выбранного заказчика setCarsBuilder(CarsBuilder carsBuilder) и
получение автомобиля по заданным параметрам getNewCar().
