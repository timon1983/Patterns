                     Шаблон проектирования Абстрактная фабрика(Abstract factory)
-С помощь данного паттерна реализован упрощенный вид программы строительства дома и магазина двумя группами рабочих.
-Группы рабочих создаются в классах ShopWorkTeam и HouseWorkTeam которые реализуют интерфейс WorkTeam.
-Для обоих объектов строительства создаются свои рабочие, которые реализуют соответствующие интерфейсы с одним
методом для выполнения своей работы.
-В классах клиентах создаем свои группы рабочих new ShopWorkTeam() и HouseWorkTeam() затем получаем самих рабочих и
даем им команду на выполнение своей работы.
-Также можно создать дополнительные группы рабочих для строительства других объектов.
